# nWatch
NKN Node Monitoring interface (desktop & mobile)

# Requirements 
Apache, PHP, PHPCurl 

# Install 
1. Rename nodes-examples.txt to nodes.txt
2. Delete the example lines
3. Add your nodes IPs and names in nodes.txt
4. Save the file, go check your dashboard.
